## スライド3: BRIEFINGフェーズ - Tier強化システム

### レイアウト
左側: 5カテゴリ図（縦並び）
右側: Tierシステム説明（2段）

### コンテンツ

**左側: 5つの装備カテゴリ**

```
🔴 消火設備 (fire)
   ├─ 役割: 攻撃・ダメージ
   └─ 例: 消火器、屋内消火栓、スプリンクラー

🟡 警報設備 (alarm)
   ├─ 役割: コスト回復（経済）
   └─ 例: 非常ベル、自動火災報知設備、放送設備

🟢 避難設備 (evacuation)
   ├─ 役割: 避難速度UP
   └─ 例: 避難はしご、誘導灯、救助袋

🔵 施設 (facility)
   ├─ 役割: バフ・サポート
   └─ 例: 連結送水管、排煙設備、非常コンセント

🟣 その他 (other)
   ├─ 役割: 特殊効果
   └─ 例: 防火扉、総合操作盤、消防機関通報設備
```

**右側上部: Tierシステム**

**3ラウンド制クイズ（各ラウンド5問）**
- ✅ 正解 → そのカテゴリのTier+1（Tier1→2→3）
- ✅ 正解 → 初期コスト+40（最大15問=+600コスト）

**Tier進化**
```
Tier1 → Tier2 → Tier3 → Tier4+ (オーバーフロー)
 ↓       ↓       ↓        ↓
基本    強化    最強    さらに強化
```

**右側下部: オーバーフロー報酬（Tier4以降）**

| 報酬タイプ | 効果 | 累積 |
|-----------|-----|------|
| 💰 コスト割引 | -10%/Tier | ✅ |
| ⚔️ 攻撃力バフ | +15%/Tier | ✅ |

**戦略例**
- **火力特化**: 消火×3でTier3+オーバーフロー（強力な攻撃力）
- **バランス型**: 消火・警報・避難×1ずつ（幅広く対応）

### デザインポイント
- カテゴリ別の色を強調（赤・黄・緑・青・紫）
- Tier進化は矢印で視覚化
- オーバーフロー報酬は金色で強調

1. デッキからカード選択

## スライド6: 勝利条件とスコアシステム

### レイアウト
上半分: 勝利・敗北条件（2列）
下半分: スコアシステム（2段）

### コンテンツ

**上半分: 勝利・敗北条件**

| ✅ 勝利条件 | ❌ 敗北条件 |
|-----------|-----------|
| **① 時間制限クリア**<br>90秒守り切る | **HP 0%到達**<br>ゲームオーバー |
| **② 避難目標達成**<br>目標人数に到達<br>• EASY: 100人<br>• NORMAL: 300人<br>• HARD: 600人 | **防衛ライン突破**<br>敵が最下段突破で<br>HP減少（ダメージ=10×敵サイズ） |
| どちらか達成で「MISSION COMPLETE」 |  |

**下半分上段: 基礎スコア計算**

```
┌─────────────────────────────────────────────────────────┐
│ 基礎スコア =                                             │
│   避難スコア（避難人数 × 100）                           │
│ + タイムボーナス（残り時間 × 5）                         │
│ + HPボーナス（HP% × 20）                                 │
│ + コストボーナス（残りコスト × 5）                       │
│ + 撃破ボーナス（撃破数 × 100）                           │
└─────────────────────────────────────────────────────────┘
```

**例**
```
避難300人(30,000) + 残り20秒(100) + HP 75%(1,500)
+ コスト500(2,500) + 撃破30体(3,000) = 基礎スコア 37,100
```

**下半分下段: バッジ倍率システム**

| バッジ | 条件 | 倍率 | 色 |
|-------|-----|------|---|
| 🏅 **NO DAMAGE** | HP 100%維持 | ×1.2 | シアン |
| 🚪 **EVACUATION COMPLETE** | 避難目標達成 | ×1.2 | 緑 |
| 💎 **ECONOMY RUN** | コスト777以上 | ×1.2 | アンバー |

**倍率の積算**
```
3つ全獲得: ×1.2 × 1.2 × 1.2 = ×1.728
```

**最終スコア計算**
```
┌─────────────────────────────────────┐
│ 最終スコア = 基礎スコア × バッジ倍率 │
└─────────────────────────────────────┘

例: 37,100 × 1.728 = 64,109点
```

### デザインポイント
- 勝利条件は緑、敗北条件は赤で色分け
- スコア計算は計算式と具体例をセットで
- バッジは色別アイコンで視覚化

### コンテンツ

**左側上段: 初心者向けTips**

### 🔰 初心者向け

✅ **基本配置**
- 消火設備を最前線（上段）に配置
- 警報設備で経済を安定化（コスト回復）
- 避難設備で早期勝利を狙う

✅ **コスト管理**
- 序盤は低コストカード（Tier1）で展開
- 中盤以降に高Tierカードを追加
- コストが足りない時は警報設備を追加

✅ **勝利のコツ**
- HP維持と避難誘導のバランスが重要
- 避難設備を配置すると勝利が早まる
- 時間制限を意識して計画的に配置

**左側下段: 上級者向けTips**

### ⭐ 上級者向け

✅ **オーバーフロー狙い**
- 一つの設備を集中強化（Tier4+）
- コスト割引と攻撃力バフで圧倒

✅ **属性相性の活用**
- 泡消火 → B火災（油火災）に特効
- ガス消火 → C火災（電気火災）に有効
- HARDでは属性相性が必須

✅ **ハイスコア狙い**
- 全バッジ達成（×1.728倍）を目指す
- 避難人数を最大化（×100点）
- HP・コスト・時間すべてを高水準で
