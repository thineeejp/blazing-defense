## 制限時間システム（v2.0新規）

### 避難設備の持続時間

避難設備は**構造物・電気設備**なので、基本的に長持ち。
一部は**永続**で、安定した基盤となる。

| 設備 | コスト | 持続時間 | 特徴 |
|------|--------|----------|------|
| 避難はしご | 40 | **永続** | 金属製の固定設備 |
| 誘導灯 | 60 | **永続** | 電気設備、常時点灯 |
| 緩降機 | 80 | **120秒** | ロープ/ベルトの消耗 |
| 救助袋 | 120 | **90秒** | 布製、摩耗がある |

### 設計思想

```
避難はしご（永続）:
- 現実: 金属製の固定設備、壊れにくい
- ゲーム: 最も基本的な避難設備、安定基盤

誘導灯（永続）:
- 現実: 電気設備、常時点灯が義務
- ゲーム: HP回復効果の継続性を保証

緩降機（120秒）:
- 現実: ロープやベルトの消耗、疲労
- ゲーム: 高効率避難なので制限あり

救助袋（90秒）:
- 現実: 布製なので摩耗がある
- ゲーム: 攻撃速度バフが強いので短めに
```

### 永続設備の戦略的価値

```
避難設備の永続設備（2種）:
🟢 避難はしご（40コスト）- 基本避難
🟢 誘導灯（60コスト）- 避難+HP回復

戦術的価値:
- 序盤に配置すれば3分間ずっと効果を発揮
- 再配置コストがかからない
- 安定した避難基盤を構築
- HP回復で長期戦に対応
```

## Tier1: 避難はしご

### 基本情報
```javascript
{
  id: 'escapeLadder',
  name: '避難はしご',
  type: 'green',
  cost: 40,
  duration: null, // 永続
  icon: <ArrowDown size={24} />,
  desc: '【避難】避難速度+0.5人/秒 [永続]',
  effect: 'evacuation',
  value: 0.5,
  rangeType: 'self',
}
```

### 性能詳細
- **効果**: 避難速度+0.5人/秒
- **攻撃力**: なし（避難設備）
- **コスト**: 40（避難設備で最安値）
- **持続時間**: **永続**（消えない）

### 特徴
- 初期から使える基本的な避難設備
- コストが安く、序盤から設置可能
- 基本避難速度（1.0人/秒）を1.5倍に強化
- 複数設置で避難速度を大幅に加速
- **永続なので1回配置すれば3分間ずっと効果**

### 運用例
- ゲーム開始直後に1〜2台設置
- 避難目標が厳しいミッションで優先配置
- 防衛に影響しない空きマスに配置
- **永続なので再配置の心配なし**

### 避難時間計算
```
基本避難速度: 1.0人/秒

避難はしご × 1台:
速度: 1.0 + 0.5 = 1.5人/秒
永続なので3分間フル稼働

ミッション別の避難時間:
図書館（100人） : 67秒（1分7秒）
百貨店（300人） : 200秒（3分20秒）※ギリギリ
化学工場（600人）: 400秒（6分40秒）※不可能

複数設置:
避難はしご × 2台 = 2.0人/秒
├─ 図書館: 50秒
├─ 百貨店: 150秒（2分30秒）
└─ 化学工場: 300秒（5分）※不可能

避難はしご × 3台 = 2.5人/秒
├─ 図書館: 40秒
├─ 百貨店: 120秒（2分）
└─ 化学工場: 240秒（4分）
```

### 永続設備としての価値
```
コスト効率計算:
避難はしご（40コスト / 永続）
3分間の避難人数: 0.5人/秒 × 180秒 = 90人

コストあたり避難: 90人 ÷ 40 = 2.25人/コスト

有限設備との比較:
緩降機（80コスト / 120秒 × 2回配置）
3分間の避難人数: 1.0人/秒 × 180秒 = 180人
総コスト: 80 × 2 = 160コスト
コストあたり避難: 180人 ÷ 160 = 1.125人/コスト

結論:
避難はしごの方がコスト効率が良い（永続の恩恵）
```

## Tier2: 誘導灯

### 基本情報
```javascript
{
  id: 'guidanceLight',
  name: '誘導灯',
  type: 'green',
  cost: 60,
  duration: null, // 永続
  icon: <Lightbulb size={24} />,
  desc: '【避難】避難速度+0.8人/秒＋HP自然回復+0.1/秒 [永続]',
  effect: 'evacuationWithRegen',
  evacuationValue: 0.8,
  regenValue: 0.1,
  rangeType: 'self',
}
```

### 性能詳細
- **効果1**: 避難速度+0.8人/秒（避難はしごの1.6倍）
- **効果2**: HP自然回復+0.1/秒
- **コスト**: 60
- **持続時間**: **永続**（消えない）

### 特徴
- 高効率な避難設備
- HP回復機能で長期戦に強い
- 避難と防衛の両立
- **永続なので安定した基盤**

### HP回復システム
```
基本HP: 100

HP回復計算（永続なので3分間フル稼働）:
誘導灯 × 1台: +0.1/秒 = +6/分 = +18/3分
誘導灯 × 2台: +0.2/秒 = +12/分 = +36/3分
誘導灯 × 3台: +0.3/秒 = +18/分 = +54/3分

戦術的価値:
- 被弾しても徐々に回復
- 長期戦で生存率UP
- 防衛失敗時の保険
- 永続なので常に回復し続ける
```

### 運用例
- **最優先で配置すべき永続設備**
- 避難速度+HP回復の複合効果
- 序盤に配置して3分間フル稼働
- 複数設置でさらに強化

### 避難時間計算
```
誘導灯 × 1台:
速度: 1.0 + 0.8 = 1.8人/秒
永続なので3分間フル稼働

ミッション別の避難時間:
図書館（100人） : 56秒
百貨店（300人） : 167秒（2分47秒）
化学工場（600人）: 333秒（5分33秒）

誘導灯 × 3台 = 3.4人/秒
├─ 図書館: 29秒
├─ 百貨店: 88秒（1分28秒）
└─ 化学工場: 176秒（2分56秒）
```

### 永続設備としての価値
```
誘導灯（60コスト / 永続）
3分間の効果:
- 避難: 0.8人/秒 × 180秒 = 144人
- HP回復: 0.1 × 180 = 18 HP

総合価値:
避難144人 + HP回復18 = 非常に高い投資効率

他の設備との比較:
救助袋（120コスト / 90秒 × 2回）
3分間の避難: 1.5人/秒 × 180秒 = 270人
3分間のHP回復: 0.2 × 180 = 36 HP
総コスト: 240コスト

コストあたり効率:
誘導灯: (144 + 18) ÷ 60 = 2.7
救助袋: (270 + 36) ÷ 240 = 1.275

結論:
誘導灯の方がコスト効率が圧倒的に良い
→ 永続設備を先に配置すべき
```

## Tier2: 緩降機

### 基本情報
```javascript
{
  id: 'descentDevice',
  name: '緩降機',
  type: 'green',
  cost: 80,
  duration: 7200, // 120秒

  effect: 'evacuation',

  rangeType: 'self',
}
```

### 性能詳細
- **効果**: 避難速度+1.0人/秒（避難はしごの2倍）
- **コスト**: 80
- **持続時間**: 120秒

### 特徴
- 最も効率的な避難速度強化（有限設備の中で）
- 設置1台で基本速度が2倍
- HP回復はないが避難特化
- **120秒と比較的長持ち**

### 運用例
- 避難速度を最優先したい時
- HP回復が不要な短期決戦
- 避難目標が非常に厳しいミッション
- **永続設備と組み合わせて使う**

### 避難時間計算
```
緩降機 × 1台:
速度: 1.0 + 1.0 = 2.0人/秒（基本速度の2倍！）
持続時間: 120秒

ミッション別の避難時間:
図書館（100人） : 50秒
百貨店（300人） : 150秒（2分30秒）
化学工場（600人）: 300秒（5分）

緩降機 × 2台 = 3.0人/秒
├─ 図書館: 33秒
├─ 百貨店: 100秒（1分40秒）
└─ 化学工場: 200秒（3分20秒）
```

### コスト効率比較
```
3分間の運用:
緩降機（80コスト / 120秒）× 2回配置 = 160コスト
避難人数: 1.0人/秒 × 180秒 = 180人
コストあたり: 180 ÷ 160 = 1.125人/コスト

避難はしご（40コスト / 永続）× 1回配置 = 40コスト
避難人数: 0.5人/秒 × 180秒 = 90人
コストあたり: 90 ÷ 40 = 2.25人/コスト

誘導灯（60コスト / 永続）× 1回配置 = 60コスト
避難人数: 0.8人/秒 × 180秒 = 144人
コストあたり: 144 ÷ 60 = 2.4人/コスト

結論:
永続設備の方がコスト効率が良い
ただし、緩降機は避難速度が最も高い（+1.0人/秒）
→ 永続設備で基盤を作り、緩降機で加速するのが最適
```

## Tier3: 救助袋

### 基本情報
```javascript
{
  id: 'rescueChute',
  name: '救助袋',
  type: 'green',
  cost: 120,

  effect: 'evacuationWithRegenAndBuff',
  evacuationValue: 1.5,
  regenValue: 0.2,
  globalSpeedBuff: 0.1,
  rangeType: 'global',
}
```

### 性能詳細
- **効果1**: 避難速度+1.5人/秒（最高値）
- **効果2**: HP自然回復+0.2/秒（誘導灯の2倍）
- **効果3**: 全タワーの攻撃速度+10%（グローバルバフ）
- **コスト**: 120
- **持続時間**: 90秒

### 特徴
- 最強の避難設備
- 避難・防衛・回復の三位一体
- グローバルバフで全タワー強化
- **高コストだが90秒と短め（布製なので摩耗）**

### グローバルバフシステム
```
効果範囲: 全てのタワー（global）

攻撃速度+10%の計算:
元の攻撃速度が40フレームのタワー
→ 40 × 0.9 = 36フレーム（約11%高速化）

救助袋 × 2台で+20%:
40 × 0.8 = 32フレーム（約25%高速化）

戦術的価値:
- 全タワーのDPS向上
- 特に高火力タワーと相性抜群
- 泡消火、不活性ガスの連射が加速
```

### 複合効果の価値
```
1回の配置（90秒）:
避難効果: 1.5人/秒 × 90秒 = 135人
HP回復効果: 0.2 × 90 = 18 HP
攻撃速度バフ: 全タワーが10%高速化

3分間（2回配置 = 240コスト）:
避難効果: 1.5人/秒 × 180秒 = 270人
HP回復効果: 0.2 × 180 = 36 HP
攻撃速度バフ: 常時（再配置で維持）

総合価値:
設置240 → 避難270人 + HP36 + 火力+10%
= 非常に高性能な複合設備
```

### 運用例
- 終盤の避難ラストスパート
- 避難と防衛の両立が必要な時
- 高火力タワーと組み合わせて総合力UP
- **90秒で消滅するので再配置が必要**

### 避難時間計算
```
救助袋 × 1台:
速度: 1.0 + 1.5 = 2.5人/秒
持続時間: 90秒

ミッション別の避難時間:
図書館（100人） : 40秒
百貨店（300人） : 120秒（2分）
化学工場（600人）: 240秒（4分）

救助袋 × 2台 = 4.0人/秒
├─ 図書館: 25秒
```

## まとめ

### 避難設備の特徴（v2.0）
✅ **勝利条件直結** - 避難人数が勝利条件
✅ **永続設備あり** - 避難はしご・誘導灯は永続
✅ **HP回復機能** - 誘導灯・救助袋で長期戦対応
✅ **攻撃速度バフ** - 救助袋で全体火力+10%

### 持続時間まとめ
| 設備 | コスト | 持続時間 | 特徴 |
|------|--------|----------|------|
| 避難はしご | 40 | **永続** | 基本、コスパ最高 |
| 誘導灯 | 60 | **永続** | HP回復付き、最優先 |
| 緩降機 | 80 | 120秒 | 高速避難 |
| 救助袋 | 120 | 90秒 | 最強、バフ付き |

### 推奨ビルド
- **初心者**: 誘導灯2台（永続）で安定基盤
- **中級者**: 永続設備+緩降機でバランス
- **上級者**: 救助袋の回転配置で避難+防衛

### 永続設備の優先度
```
1. 誘導灯（避難+HP回復、最優先）
2. 避難はしご（安価、コスパ最高）
3. 連結送水管（バフ、消火設備と組み合わせ）
4. 非常用エレベーター（コスト割引）
```

### 注意点
⚠️ 永続設備を先に配置するのが最もコスト効率が良い
⚠️ 有限設備（緩降機・救助袋）は再配置コストを計算
⚠️ 化学工場（600人）は永続設備だけでは厳しい
