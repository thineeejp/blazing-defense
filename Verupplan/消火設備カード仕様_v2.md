```
Tier1（初期アンロック）
└─ 消火器

Tier2（1回目選択でアンロック）
├─ 屋内消火栓設備

Tier3（2回目選択でアンロック）

└─ 不活性ガス消火設備（全体攻撃）
```

## 制限時間システム（v2.0新規）

### 消火設備の持続時間

消火設備は**消火剤が有限**という現実を反映し、全て有限の持続時間を持つ。

| 設備 | コスト | 持続時間 | 3分間配置数 | 総コスト |
|------|--------|----------|-------------|----------|
| 消火器 | 30 | **45秒** | 4回 | 120 |
| 屋内消火栓 | 80 | **90秒** | 2回 | 160 |
| スプリンクラー | 90 | **120秒** | 2回 | 180 |
| 泡消火設備 | 120 | **60秒** | 3回 | 360 |
| 不活性ガス | 100 | **45秒** | 4回 | 400 |

### 設計思想

```
消火器（45秒）: 
- 現実: 10型消火器の放射時間は約15秒
- ゲーム: コスト効率が高いので短めに

屋内消火栓（90秒）:
- 現実: 水源に接続しているが、操作する人間の疲労
- ゲーム: 中程度の効率なので標準的

スプリンクラー（120秒）:
- 現実: 自動散水、水源が豊富
- ゲーム: DPS効率が低いので長めに補完

泡消火設備（60秒）:
- 現実: 泡原液タンクは有限
- ゲーム: 高火力＋B火災特効なので短めに

不活性ガス（45秒）:
- 現実: ガスボンベは有限、放出後は拡散
- ゲーム: 全体攻撃が非常に強いので短めに
```

## Tier1: 消火器

### 基本情報
```javascript
{
  id: 'extinguisher',
  name: '消火器',
  type: 'red',
  cost: 30,
  duration: 2700, // 45秒
  icon: <Shield size={24} />,
  desc: '【消火】周囲1マス(3×3)へ短距離散水 [45秒]',
  rangeType: 'surround',
  power: 2,
  speed: 40,
  damageType: 'water',
}
```

### 性能詳細
- **攻撃範囲**: 周囲3×3マス（surround）
- **攻撃力**: 2
- **攻撃速度**: 40フレームごと（約0.67秒）
- **ダメージタイプ**: water
- **コスト**: 30（最安値）
- **持続時間**: 45秒

### 特徴
- 初期から使える基本的な消火設備
- コストが安く序盤の展開に最適
- 範囲攻撃で複数の敵に対応可能
- B火災（油火災）には効果が低い（0.1倍）
- **45秒で消滅するため、再配置が必要**

### DPS計算
```
攻撃力: 2
攻撃間隔: 40フレーム = 0.67秒
DPS: 2 ÷ 0.67 = 3.0

45秒間の総ダメージ:
3.0 × 45 = 135ダメージ

コスト効率:
135ダメージ ÷ 30コスト = 4.5ダメージ/コスト
```

### 運用例
- 序盤の防衛ライン付近に複数設置
- 45秒後に消滅 → 再配置 or 別タワーに切り替え
- 連結送水管の周囲に配置してバフを受ける

### 3分間の運用コスト
```
1回配置: 30コスト / 45秒
3分間: 4回配置 = 120コスト

実質コスト効率:
永続だった場合の4倍のコストがかかる
→ バランス調整の結果
```

## Tier2: 屋内消火栓設備

### 基本情報
```javascript
{
  id: 'indoorHydrant',
  name: '屋内消火栓設備',
  type: 'red',
  cost: 80,
  duration: 5400, // 90秒
  icon: <Droplets size={24} />,
  desc: '【消火】前方縦1列に強力な放水 [90秒]',
  rangeType: 'line',
  power: 4,
  speed: 50,
  damageType: 'water',
  knockback: 0.3,
}
```

### 性能詳細
- **攻撃範囲**: 縦1列（line）
- **攻撃力**: 4（消火器の2倍）
- **攻撃速度**: 50フレームごと（約0.83秒）
- **ダメージタイプ**: water
- **特殊効果**: ノックバック0.3（敵を押し戻す）
- **コスト**: 80
- **持続時間**: 90秒

### 特徴
- 縦1列の敵をまとめて攻撃
- 高火力で大型の敵にも有効
- ノックバックで敵の到達を遅延
- B火災には注意（water属性のため）
- **90秒と比較的長持ち**

### DPS計算
```
攻撃力: 4
攻撃間隔: 50フレーム = 0.83秒
DPS: 4 ÷ 0.83 = 4.8

90秒間の総ダメージ:
4.8 × 90 = 432ダメージ

コスト効率:
432ダメージ ÷ 80コスト = 5.4ダメージ/コスト
```

### 運用例
- 敵の進行ルート上に設置
- 大型化した敵を押し戻す
- 中盤以降の主力として活躍
- 90秒持つので2回配置で3分間カバー可能

## Tier3: 泡消火設備

### 基本情報
```javascript
{
  id: 'foamSystem',
  name: '泡消火設備',
  type: 'red',
  cost: 120,
  duration: 3600, // 60秒
  icon: <ShieldAlert size={24} />,
  desc: '【特効】周囲1マスに大ダメージ（B火災に強い）[60秒]',
  rangeType: 'surround',
  power: 8,
  speed: 60,
  damageType: 'foam',
}
```

### 性能詳細
- **攻撃範囲**: 周囲3×3マス（surround）
- **攻撃力**: 8（基本値）
- **攻撃速度**: 60フレームごと（約1秒）
- **ダメージタイプ**: foam
- **B火災への倍率**: 2.0倍（16ダメージ）
- **コスト**: 120
- **持続時間**: 60秒

### 特徴
- B火災（油火災）に特効
- 高火力で大型の敵も撃破可能
- 周囲攻撃で複数の敵に対応
- water属性の弱点を補完
- **泡原液が有限なので60秒と短め**

### DPS計算
```
攻撃力: 8（B火災は16）
攻撃間隔: 60フレーム = 1秒
DPS: 8.0（B火災は16.0）

60秒間の総ダメージ:
通常: 8 × 60 = 480ダメージ
B火災: 16 × 60 = 960ダメージ

コスト効率:
通常: 480 ÷ 120 = 4.0ダメージ/コスト
B火災: 960 ÷ 120 = 8.0ダメージ/コスト
```

### 運用例
- B火災が多いミッションで優先配置
- 重要な防衛ポイントに設置
- 連結送水管のバフと相性が良い
- **60秒で消滅→再配置のサイクルが必要**

### 3分間の運用
```
3分間: 3回配置 = 360コスト
総ダメージ: 480 × 3 = 1,440ダメージ（通常）
         : 960 × 3 = 2,880ダメージ（B火災）

高コストだがB火災戦では必須
```

## Tier3: 不活性ガス消火設備

### 基本情報
```javascript
{
  id: 'inertGasSystem',
  name: '不活性ガス消火設備',
  type: 'red',
  cost: 100,
  duration: 2700, // 45秒
  icon: <Wind size={24} />,
  desc: '【特効】全画面へ持続ダメージ(DoT) [45秒]',
  rangeType: 'global',
  power: 0.4,
  speed: 5,
  damageType: 'gas',
}
```

### 性能詳細
- **攻撃範囲**: 全画面（global）
- **攻撃力**: 0.4（毎フレーム）
- **攻撃速度**: 5フレームごと（約0.08秒、超高速）
- **ダメージタイプ**: gas
- **C火災への倍率**: 1.5倍
- **コスト**: 100
- **持続時間**: 45秒

### 特徴
- 全ての敵に持続的にダメージ
- 設置するだけで全体を攻撃
- C火災（電気火災）に特効
- 数が多い敵に対して非常に有効
- **ガスボンベが有限なので45秒と短め**

### DPS計算
```
攻撃力: 0.4
攻撃間隔: 5フレーム = 0.083秒
1秒間の攻撃回数: 12回
DPS: 0.4 × 12 = 4.8（全体）

C火災: 0.4 × 1.5 × 12 = 7.2

45秒間の総ダメージ:
通常: 4.8 × 45 = 216ダメージ（全体に）
C火災: 7.2 × 45 = 324ダメージ

コスト効率:
216ダメージ ÷ 100コスト = 2.16ダメージ/コスト
※ただし全体攻撃なので敵の数だけ効果倍増
```

### 運用例
- 敵の数が多い終盤に設置
- 複数設置で持続ダメージを重ねる
- 他のタワーと組み合わせて総合火力UP
- **45秒と短いので、4回配置が必要**

### 3分間の運用
```
3分間: 4回配置 = 400コスト

敵10体に対する効果:
216 × 10 = 2,160ダメージ相当/回
3分間: 8,640ダメージ相当

非常に強力だがコストも高い
```

## 消火設備の戦略的組み合わせ

### パターン1: 序盤重視型
```
Tier1のみ（初期状態）
├─ 消火器 × 4台（120コスト / 45秒ごとに再配置）
└─ 連結送水管 × 1台（60コスト / 永続バフ）
総コスト: 180（初期配置）

戦術: 
- 連結送水管の周囲に消火器を配置
- 45秒ごとに消火器を再配置
- バフされた消火器（2.6ダメージ）で序盤を凌ぐ
```

### パターン2: 中盤バランス型
```
Tier1 + Tier2（ラウンド1で消火設備選択）
├─ 消火器 × 2台（60コスト / 45秒）
├─ 屋内消火栓 × 1台（80コスト / 90秒）
└─ スプリンクラー × 1台（90コスト / 120秒）
総コスト: 230（初期配置）

戦術:
- スプリンクラーは120秒持つので安定
- 屋内消火栓は90秒でノックバック
- 消火器は45秒で消えるので適宜再配置
```

### パターン3: 終盤特化型
```
Tier1 + Tier2 + Tier3（ラウンド1,2で消火設備選択）
├─ 泡消火設備 × 1台（120コスト / 60秒）
├─ 不活性ガス × 2台（200コスト / 45秒）
├─ 屋内消火栓 × 2台（160コスト / 90秒）
└─ スプリンクラー × 1台（90コスト / 120秒）
総コスト: 570（初期配置）

戦術:
- 不活性ガスで全体攻撃（45秒で再配置）
- 泡消火でB火災対策（60秒で再配置）
- 屋内消火栓・スプリンクラーは長持ち
```

### パターン4: 回転型（制限時間を活かす）
```
0〜45秒: 
├─ 消火器 × 3台
├─ 不活性ガス × 1台
└─ 連結送水管 × 1台（永続）

45〜90秒:
├─ 消火器が消滅 → 屋内消火栓に切り替え
├─ 不活性ガスが消滅 → 再配置
└─ 連結送水管は維持

90〜135秒:
├─ 屋内消火栓は維持
├─ 不活性ガスが消滅 → 泡消火設備に切り替え
└─ 連結送水管は維持

135〜180秒:
├─ 屋内消火栓が消滅 → 再配置 or スプリンクラー
├─ 泡消火設備が消滅 → 再配置
└─ 最終防衛

戦術:
- 時間経過に応じてタワーを切り替える
- 状況に応じた最適なタワーを選択
- 「設置して放置」ではなく、常に判断が必要
```

## 他設備とのシナジー

### 🔵 消火活動上必要な施設との組み合わせ

**連結送水管（buffPower）- 永続**
```
効果: 周囲のタワーの攻撃力+30%

消火器（2 → 2.6）
屋内消火栓（4 → 5.2）
泡消火（8 → 10.4、B火災で20.8！）

戦術: 
- 連結送水管は永続なので先に配置
- 消火設備は消えても、連結送水管は残る
- 再配置した消火設備も自動的にバフを受ける
```

**非常コンセント（buffSpeed）- 120秒**
```
効果: 全タワーの攻撃速度+20%

スプリンクラー（30f → 24f）
不活性ガス（5f → 4f）

戦術: 
- 非常コンセントは120秒持つ
- 攻撃速度の速いタワーと相性抜群
```

### 🟡 警報設備との組み合わせ

**経済設備でコスト確保**
```
消火設備は再配置にコストがかかる
→ 警報設備で継続的にコスト回収が必須

例:
非常ベル × 2台（60コスト / 60秒）= 1.2コスト/秒
↓
3分間で216コスト回収
↓
消火器4回再配置（120コスト）を賄える
```

## 火災タイプ別の対策

### A火災（普通火災）
```
推奨: 全ての消火設備が等倍で有効
最適: スプリンクラー（高速連射、120秒持続）
```

### B火災（油火災）
```
危険: water属性は0.1倍に減衰
最適: 泡消火設備（2.0倍）[60秒]
次点: 不活性ガス（等倍）[45秒]

注意: 泡消火は60秒で消えるので、
      B火災が多いミッションでは再配置コストを計算
```

### C火災（電気火災）
```
推奨: 不活性ガス（1.5倍）[45秒]
注意: 45秒と短いので、再配置を忘れずに
```

## まとめ

### 消火設備の特徴
✅ **攻撃力が高い** - ゲームの主力となる火力源
✅ **多様な範囲** - surround/line/wide/globalで状況に対応
✅ **属性相性** - B火災とC火災に対する特効が存在
✅ **消耗品** - 全て有限の持続時間（消火剤は切れる）

### 持続時間まとめ
| 設備 | 持続時間 | 理由 |
|------|----------|------|
| 消火器 | 45秒 | 薬剤が切れる、コスト効率調整 |
| 屋内消火栓 | 90秒 | 水源接続、標準的 |
| スプリンクラー | 120秒 | 自動散水、DPS低いので補完 |
| 泡消火 | 60秒 | 泡原液有限、高火力なので短め |
| 不活性ガス | 45秒 | ガスボンベ有限、全体攻撃強い |

### 推奨ビルド
- **初心者**: 消火器+連結送水管（永続）で安定
- **中級者**: スプリンクラー（120秒）を軸に
- **上級者**: 不活性ガスの回転配置で火力最大化

### 注意点
⚠️ 全ての消火設備は消滅する（再配置コストを計算）
⚠️ B火災が多いミッションでは泡消火必須（60秒×3回=360コスト）
⚠️ 経済設備との併用が必須
