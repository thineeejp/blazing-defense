## 制限時間システム（v2.0新規）

### 施設設備の持続時間

施設は**建物に付随する設備**なので、基本的に長持ち。
一部は**永続**で、安定したバフを提供。

| 設備 | コスト | 持続時間 | 特徴 |
|------|--------|----------|------|
| 連結送水管 | 60 | **永続** | 建物の配管設備 |
| 非常コンセント | 100 | **120秒** | 電源設備だが過負荷 |
| 排煙設備 | 150 | **90秒** | モーター過熱 |

### 設計思想

```
連結送水管（永続）:
- 現実: 建物の配管設備、壊れにくい
- ゲーム: 配置場所が重要なバフ、永続で安定

非常コンセント（120秒）:
- 現実: 電源設備だが、過負荷で停止する可能性
- ゲーム: グローバルバフなので制限あり

排煙設備（90秒）:
- 現実: 排煙ファンのモーター過熱、フィルタ詰まり
- ゲーム: 強効果なので短めに
```

## バランス調整（v2.0）

### 排煙設備の効果変更

**旧仕様の問題:**
```
排煙設備（旧）:
- 全エリアの敵 -20%移動速度
- 全タワーの攻撃力 +15%

問題点:
1. 「煙を排出する」設備なのに、敵の移動を遅くするのは違和感
2. 減速+攻撃力の複合効果が強すぎる
3. 複数設置で効果が過剰に
```

**新仕様（v2.0）:**
```
排煙設備（新）:
- 全エリアの敵 -15%移動速度（弱体化: -20% → -15%）
- 避難速度 +0.5人/秒（新規追加）

設計理由:
1. 排煙で視界確保 → 避難しやすくなる（避難速度UP）
2. 煙を排出しても敵（火災）が多少遅くなるのは許容範囲
3. 攻撃力バフを削除して複合効果を抑制
4. 減速効果を-15%に弱体化
```

### 変更の影響

```
旧仕様での問題:
排煙設備 × 3台 = -60%速度 + +45%攻撃力
→ 実質火力が約2.6倍に（強すぎる）

新仕様:
排煙設備 × 3台 = -45%速度 + 避難+1.5人/秒
→ 減速効果は控えめ、避難支援に特化
→ 攻撃力バフは連結送水管に任せる
```

## Tier1: 連結送水管

### 基本情報
```javascript
{
  id: 'standpipe',
  name: '連結送水管',
  type: 'blue',
  cost: 60,
  duration: null, // 永続
  icon: <Waves size={24} />,
  desc: '【施設】周囲3×3のタワーの攻撃力+30% [永続]',
  effect: 'buffPower',
  range: 'surround',
  value: 0.3,
  rangeType: 'surround',
}
```

### 性能詳細
- **効果**: 周囲3×3マスのタワーの攻撃力+30%
- **範囲**: 自分を中心とした3×3マス
- **攻撃力**: なし（バフ設備）
- **コスト**: 60
- **持続時間**: **永続**（消えない）

### 特徴
- 初期から使える基本的なバフ設備
- 周囲のタワーの火力を大幅強化
- 高火力タワーと組み合わせて効果最大化
- 配置場所が戦術の核
- **永続なので最優先で配置すべき**

### バフ計算
```
攻撃力+30%の効果:

消火器（攻撃力2）:
2 × 1.3 = 2.6

屋内消火栓（攻撃力4）:
4 × 1.3 = 5.2

泡消火設備（攻撃力8）:
8 × 1.3 = 10.4
→ B火災で 10.4 × 2.0 = 20.8！

不活性ガス（攻撃力0.4/フレーム）:
0.4 × 1.3 = 0.52/フレーム
DPS: 4.8 → 6.24（+30%）
```

### バフ範囲の可視化
```
[グリッド配置例]
Row 3: [泡消火] [屋内消火栓] [  ]
Row 4: [消火器] [連結送水管] [消火器]  ← 中心（永続）
Row 5: [  ] [スプリンクラー] [  ]

バフ対象（周囲3×3）:
├─ 泡消火: 8 → 10.4
├─ 屋内消火栓: 4 → 5.2
├─ 消火器 × 2: 2 → 2.6
└─ スプリンクラー: 1.5 → 1.95

永続の利点:
- 連結送水管は消えない
- 周囲の消火設備は消えても、再配置で再びバフを受ける
- 「バフの中心」として機能し続ける
```

### 運用例
- **最優先で配置すべき永続設備**
- 防衛ラインの中央に設置
- 高火力タワーを周囲に配置
- 消火設備が消えても、連結送水管は残る

### 永続設備としての価値
```
連結送水管（60コスト / 永続）
3分間の効果:
- 周囲8マスに常時+30%バフ
- 再配置コストゼロ

仮に周囲5台のタワーがバフを受ける場合:
平均攻撃力4 × 5台 × 0.3 = 6.0相当の火力増加
3分間フル稼働 = 非常に高い投資効率
```

### 性能詳細
- **効果1**: 全タワーの攻撃速度+20%（グローバル）

### 特徴
- グローバルバフで全体強化
- 攻撃速度特化のバフ設備
- HP回復機能も付属
- 高速タワーとの相性抜群
- **120秒と比較的長持ち**

### 攻撃速度バフの計算
```
攻撃速度+20%（攻撃間隔が0.8倍に）:

スプリンクラー（30フレームごと）:
30 × 0.8 = 24フレーム
DPS: 1.5 ÷ 0.5秒 = 3.0 → 3.75（+25%）

不活性ガス（5フレームごと）:
5 × 0.8 = 4フレーム
DPS: 0.4 × 12回 = 4.8 → 6.0（+25%）

屋内消火栓（50フレームごと）:
50 × 0.8 = 40フレーム
DPS: 4 ÷ 0.83秒 = 4.8 → 6.0（+25%）

結論:
全てのタワーのDPSが約+25%向上！
```

### 救助袋との比較
```
┌────────────────┬──────────┬──────────┐
│                │ 救助袋   │ 非常コン │
├────────────────┼──────────┼──────────┤
│ コスト         │ 120      │ 100      │
│ 攻撃速度バフ   │ +10%     │ +20%     │
│ 効果範囲       │ グローバル│ グローバル│

結論:
✅ 攻撃速度重視 → 非常コンセント（+20%、120秒）
✅ 避難も欲しい → 救助袋
✅ 両方設置で → 攻撃速度+30%の超強化！
```

### HP回復効果
```
HP回復: +0.15/秒
120秒間の総回復: 0.15 × 120 = 18 HP

3分間（2回配置）:
0.15 × 180 = 27 HP

誘導灯（+0.1/秒 / 永続）より1.5倍速いが、
誘導灯は永続なので総回復量は誘導灯の方が多い
```

### 運用例
- スプリンクラー・不活性ガスと組み合わせ
- 終盤の総火力底上げ
- HP回復で長期戦対応
- **120秒持つので2回配置で3分間カバー**

## Tier3: 排煙設備

### 基本情報（v2.0で変更）
```javascript
{
  id: 'smokeControl',
  name: '排煙設備',
  type: 'blue',
  cost: 150,
  duration: 5400, // 90秒
  icon: <Wind size={24} />,
  desc: '【施設】全敵-15%速度＋避難+0.5人/秒 [90秒]',
  effect: 'globalSlowWithEvacuation',
  slowValue: 0.15,  // v2.0: 0.2 → 0.15に弱体化
  evacuationValue: 0.5,  // v2.0: 新規追加
  rangeType: 'global',
}
```

### 性能詳細（v2.0で変更）
- **効果1**: 全エリアの敵の移動速度-15%（v2.0: -20%から弱体化）
- **効果2**: 避難速度+0.5人/秒（v2.0: 新規追加）
- **攻撃力バフ**: なし（v2.0: 削除）
- **コスト**: 150
- **持続時間**: 90秒

### 変更理由
```
旧仕様の問題:
1. 排煙設備は「煙を排出」する設備
2. 「敵を遅くする」のは違和感があった
3. 減速+攻撃力バフの複合が強すぎた

新仕様の設計:
1. 排煙で視界確保 → 避難しやすい（避難速度UP）
2. 減速効果は控えめに残す（-20% → -15%）
3. 攻撃力バフは削除（連結送水管に任せる）
4. 「避難支援」にフォーカス
```

### 移動速度デバフ
```
効果範囲: 全エリア（全ての列・全ての敵）

敵の移動速度-15%:
基本速度 1.0 → 0.85
高速敵 1.5 → 1.275
低速敵 0.7 → 0.595

滞在時間: 1.0 ÷ 0.85 = 1.18倍

複数設置時:
排煙設備 × 2台: -30% → 滞在1.43倍
排煙設備 × 3台: -45% → 滞在1.82倍

戦術的価値:
- 全ての敵の滞在時間が延びる
- 攻撃回数が増える
- 配置場所に関係なく効果発動
```

### 避難効果（v2.0新規）
```
避難速度: +0.5人/秒
90秒間の避難人数: 0.5 × 90 = 45人

3分間（2回配置）:
0.5 × 180 = 90人避難

避難設備との組み合わせ:
誘導灯（永続 / +0.8人/秒）+ 排煙設備（90秒 / +0.5人/秒）
= 合計 +1.3人/秒

戦術的価値:
- 避難と減速の両立
- 「視界確保で避難しやすい」という現実を反映
```

### 運用例
- 空きマスに自由に配置（グローバル効果）
- 避難目標達成の補助に
- 減速効果で全体の攻撃回数UP
- **90秒で消滅→再配置のサイクル**

### 旧仕様との比較
```
旧仕様（v1.0）:
├─ 移動速度: -20%
├─ 攻撃力: +15%
├─ 避難: なし
└─ 複数設置で非常に強力

新仕様（v2.0）:
├─ 移動速度: -15%（弱体化）
├─ 攻撃力: なし（削除）
├─ 避難: +0.5人/秒（新規）
└─ バランス改善

結論:
攻撃特化 → 避難支援にシフト
現実の「排煙で視界確保→避難促進」を反映
強すぎた効果を適正化
```

## まとめ

### 消火活動上必要な施設の特徴（v2.0）
✅ **バフ専門** - 他タワーを強化する唯一の種別
✅ **永続設備あり** - 連結送水管は永続
✅ **攻撃力バフ** - 連結送水管（周囲）
✅ **攻撃速度バフ** - 非常コンセント（グローバル）
✅ **減速+避難** - 排煙設備（v2.0で変更）

### 持続時間まとめ
| 設備 | コスト | 持続時間 | 効果 |
|------|--------|----------|------|
| 連結送水管 | 60 | **永続** | 周囲+30%攻撃力 |
| 非常コンセント | 100 | 120秒 | 全体+20%速度+HP回復 |
| 排煙設備 | 150 | 90秒 | 全体-15%速度+避難+0.5 |

### v2.0での変更点
- **排煙設備**: 効果を「減速+避難」に変更
  - 移動速度デバフ: -20% → -15%（弱体化）
  - 攻撃力バフ: 削除
  - 避難速度: +0.5人/秒（新規追加）
- **現実整合性**: 「排煙で視界確保→避難促進」を反映

### 推奨ビルド
- **初心者**: 連結送水管（永続）で安定バフ
- **中級者**: 非常コンセントで攻撃速度UP
- **上級者**: 排煙設備で避難+減速の複合運用

### 永続設備の優先度
```
1. 連結送水管（攻撃力バフ、配置重要）
2. 誘導灯（避難+HP回復）
3. 避難はしご（基本避難）
4. 非常用エレベーター（コスト割引）
```

### 注意点
⚠️ 連結送水管を先に配置（永続）
⚠️ 排煙設備は90秒で消滅（再配置コストを計算）
⚠️ 消火設備との組み合わせで真価を発揮
