## v2.0での主な変更点

### 1. Tier1の初期解放

```
旧仕様（v1.0）:
- その他: Tier1も完全ロック
- BRIEFINGで選択しないと使えない

新仕様（v2.0）:
- その他: Tier1が初期アンロック
- 他の4種別と同様に使える

変更理由:
1. 「その他」を使うハードルが高すぎた
2. Tier3（防災センター）を出すには3回選択が必要
3. 他種別のTier1が使えなくなる問題
4. 初期解放で戦略の幅が広がる
```

### 2. 非常用エレベーターの永続化

```
旧仕様（v1.0）:
- 非常用エレベーター: 持続時間なし（設定漏れ）

新仕様（v2.0）:
- 非常用エレベーター: 永続
- 建物設備なので壊れにくい
- コスト割引効果は早期配置ほど有利なので永続が適切
```

## Tier1: 防火戸

### 基本情報
```javascript
{
  id: 'fireDoor',
  name: '防火戸',
  type: 'purple',
  cost: 120,
  duration: 300, // 5秒
  icon: <DoorClosed size={24} />,
  desc: '【特殊】設置から5秒間、横1列の敵を完全停止→消滅',
  effect: 'rowBlockTimed',
  blockDuration: 300,
  selfDestruct: true,
  rangeType: 'row',
}
```

### 性能詳細
- **効果**: 設置した横1行の敵を5秒間完全停止
- **持続時間**: 5秒（設置から消滅まで）
- **攻撃力**: なし（特殊設備）
- **コスト**: 120

### 特徴
- 設置した瞬間から5秒間、横1列を封鎖
- 5秒後に自動消滅（マスが空く）
- 使い捨ての時間稼ぎ設備
- 消滅後は別のタワーを配置可能
- **v2.0: 初期から使える**

### 動作メカニズム
```
設置時（0秒）:
Row 4に防火戸を設置
↓
即座に発動: Row 4の全ての敵が停止開始
↓
0〜5秒: Row 4を通る全ての敵が停止
↓
5秒後: 防火戸が消滅、マスが空きに
↓
以降: Row 4の敵は通常通り移動

画面表示（設置時）:
┌─────────────────────────┐
│ 🚪 防火戸閉鎖！         │
│ Row 4 を5秒間封鎖       │
└─────────────────────────┘

カウントダウン表示:
[🚪] 残り4.2秒
[🚪] 残り3.1秒
[🚪] 残り2.0秒
[🚪] 残り1.0秒
[💨] 消滅！
```

### 戦術的価値
```
5秒間の停止 = 攻撃機会300フレーム分

スプリンクラー（30fごと）の場合:
通常: 10回攻撃
停止中: +10回攻撃（合計20回、2倍！）

不活性ガス（5fごと）の場合:
通常: 60回攻撃
停止中: +60回攻撃（合計120回、2倍！）

全タワー5台が攻撃できる場合:
総ダメージが2倍 = 5台分の追加ダメージ
= 約200コスト相当の価値

結論:
コスト120で200相当の効果
さらに5秒後にマスが空いて再利用可能
```

### 運用例
- 緊急時間稼ぎ（敵が防衛ラインに迫る時）
- 波状防衛（時間差で複数配置）
- マス再利用戦術（5秒後に別タワー配置）

## Tier1: 非常用エレベーター

### 基本情報（v2.0で永続化）
```javascript
{
  id: 'emergencyElevator',
  name: '非常用エレベーター',
  type: 'purple',
  cost: 100,
  duration: null, // v2.0: 永続
  icon: <MoveVertical size={24} />,
  desc: '【特殊】全タワーの攻撃速度+15%＋配置コスト-10% [永続]',
  effect: 'firefighterSupport',
  globalSpeedBuff: 0.15,
  costReduction: 0.1,
  rangeType: 'global',
}
```

### 性能詳細
- **効果1**: 全タワーの攻撃速度+15%（消防隊の迅速な活動）
- **効果2**: 全タワーの配置コスト-10%（迅速な部隊展開）
- **コスト**: 100
- **持続時間**: **永続**（v2.0で変更）

### 特徴
- 消防隊の活動を支援する設備
- 迅速な移動 = 攻撃速度UP
- 迅速な展開 = 配置コストDOWN
- **永続なので早期配置ほど有利**
- **v2.0: 初期から使える**

### コンセプト
```
リアルの非常用エレベーター:
「消防隊が高層階へ迅速に到達し、
 効率的に消火活動を行うための設備」

ゲーム内での解釈:
✓ 迅速な移動 → 攻撃速度+15%
✓ 効率的な展開 → 配置コスト-10%
✓ 消防隊の活動支援 → 全タワーを強化
```

### 攻撃速度+15%の効果
```
スプリンクラー（30フレームごと）:
30 × 0.85 = 25.5フレーム
DPS: 3.0 → 3.53（+18%）

不活性ガス（5フレームごと）:
5 × 0.85 = 4.25フレーム
DPS: 4.8 → 5.65（+18%）

結論:
全タワーの火力が約+18%向上
永続なので3分間ずっと効果
```

### 配置コスト-10%の効果
```
コスト割引の適用:

消火器（30コスト）:
30 × 0.9 = 27コスト

泡消火設備（120コスト）:
120 × 0.9 = 108コスト

総コスト500で配置する場合:
通常: 500コスト分のタワー
割引後: 約555コスト分のタワー

3分間で1,000コスト配置する場合:
節約: 100コスト（非常用エレベーターのコスト分）
→ 実質タダで攻撃速度+15%を獲得
```

### 永続設備としての価値（v2.0）
```
非常用エレベーター（100コスト / 永続）

早期配置の利点:
- 以降の全タワーが-10%コスト
- 以降の全タワーが+15%攻撃速度

開幕で配置した場合:
3分間で1,000コスト配置 → 100コスト節約
3分間で全タワー+15%攻撃速度

結論:
最も早く配置すべき永続設備の1つ
```

### 運用例
- **開幕で最優先配置**
- 以降のタワー配置が全て効率化
- 攻撃速度+15%が3分間持続

## Tier2: パッケージ型自動消火設備

### 基本情報
```javascript
{
  id: 'packageFireSystem',
  name: 'パッケージ型自動消火設備',
  type: 'purple',
  cost: 120,
  duration: 3600, // 60秒
  icon: <Package size={24} />,
  desc: '【特殊】周囲3×3にDoT1.0/秒＋移動速度-30% [60秒]',
  effect: 'areaDotWithSlow',
  dotDamage: 1.0,
  slowValue: 0.3,
  rangeType: 'surround',
}
```

### 性能詳細
- **効果1**: 周囲3×3マスに持続ダメージ（DoT）1.0/秒
- **効果2**: 周囲3×3マスの敵の移動速度-30%
- **コスト**: 120
- **持続時間**: 60秒

### 特徴
- 持続ダメージ + 減速の複合効果
- 範囲内の全ての敵に効果
- 設置するだけで継続的にダメージ
- 減速効果で滞在時間も延長
- **消火剤が有限なので60秒と短め**

### DoT + 減速の相乗効果
```
移動速度-30%:
基本速度1.0 → 0.7
滞在時間: 1.43倍

DoTの実質ダメージ:
1.0/秒 × 1.43倍 = 1.43相当

60秒間の効果:
DoT: 1.0 × 60 = 60ダメージ
減速による追加効果: 他タワーの攻撃回数も増加

複数設置時:
パッケージ型 × 2台（範囲重複）
├─ DoT: 2.0/秒
├─ 減速: -60%（移動速度0.4、滞在時間2.5倍）
└─ 実質: 2.0 × 2.5 = 5.0相当
```

### 運用例
- 防衛ラインの中央に設置
- 複数の敵を同時に攻撃+減速
- 他のタワーの攻撃機会増加
- **60秒で消滅→再配置のサイクル**

```

### 性能詳細

- **コスト**: 50（自動火災報知設備より安い）
- **持続時間**: 90秒
- **攻撃力**: なし（経済設備）

### 特徴
- 自動火災報知設備の廉価版

- 早期警戒機能はないが純粋な経済設備

### 投資回収計算
```
初期コスト: 50
回収量: 1.0/秒 = 60/分

1回の配置（90秒）:
回収量: 1.0 × 90 = 90コスト
純利益: 90 - 50 = +40コスト

3分間の運用:
配置回数: 2回
総コスト: 50 × 2 = 100コスト
総回収: 90 × 2 = 180コスト
純利益: 180 - 100 = +80コスト

警報設備との比較:
自動火災報知（60コスト / 90秒）: 純利益+96
特小自火報（50コスト / 90秒）: 純利益+80

結論:
警報設備を選ばなかった場合の経済手段
```

### 運用例
- 「その他」選択時の経済確保
- 序盤の資金繰り
- 警報設備と併用して経済力UP

### 攻撃力+攻撃速度の複合バフ
```
泡消火設備（攻撃力8、60フレームごと）の場合:

攻撃力: 8 × 1.2 = 9.6
攻撃速度: 60 × 0.8 = 48フレーム

→ 元のDPS（8 ÷ 1秒 = 8）から1.5倍！

B火災の場合:
12 × 2.0 = 24 DPS
→ 元の16から1.5倍

結論:
攻撃力+攻撃速度の両方を強化
→ DPSが約1.5倍に向上
```

```
攻撃面:
全タワー10台 × 平均DPS 4
→ 40 DPS → 60 DPS（+50%）
120秒間の追加ダメージ: 20 × 120 = 2,400

避難面:
+1.0人/秒
→ 緩降機1台分の避難力
120秒間の避難: 120人

回復面:
+0.3/秒
→ 120秒で+36 HP

総合価値:
設置200 → 攻撃+50% + 避難120人 + HP+36
= 圧倒的な投資効率

|------|--------|----------|------|
| 防火戸 | 120 | 5秒 | 緊急停止→消滅 |

- **初心者**: 非常用エレベーター（永続）を開幕配置
