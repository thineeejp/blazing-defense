## 制限時間システム（v2.0新規）

### 警報設備の持続時間

警報設備は**電子機器・電気設備**なので、消火設備より長持ちする傾向。

| 設備 | コスト | 持続時間 | 3分間配置数 | 経済効率 |
|------|--------|----------|-------------|----------|
| 非常ベル | 30 | **60秒** | 3回 | 調整後 |
| 自動火災報知 | 60 | **90秒** | 2回 | 標準 |
| 放送設備 | 100 | **120秒** | 2回 | 最高 |
| 消防機関通報 | 120 | **特殊** | - | 変身型 |

### バランス調整（v2.0）

```
旧バージョンの問題:
非常ベル（25コスト / 永続）= 経済効率4.3倍
→ 他の経済設備より圧倒的に強かった

v2.0での調整:
1. 非常ベルのコストを25→30に引き上げ
2. 持続時間60秒を設定
3. 結果: 経済効率が他と同程度に

調整後の経済効率比較:
非常ベル: 純利益 +33（3分間）
自動火災報知: 純利益 +96（3分間）
放送設備: 純利益 +280（3分間）
→ 上位設備ほど効率が良い、正常なバランスに
```

## Tier1: 非常ベル

### 基本情報
```javascript
{
  id: 'emergencyBell',
  name: '非常ベル',
  type: 'yellow',
  cost: 30,  // v2.0: 25→30に調整
  duration: 3600, // 60秒
  icon: <Bell size={24} />,
  desc: '【警報】コスト回復+0.6/秒 [60秒]',
  effect: 'economy',
  value: 0.6,
  rangeType: 'self',
}
```

### 性能詳細
- **効果**: コスト回復量+0.6/秒
- **攻撃力**: なし（経済設備）
- **コスト**: 30（v2.0で25から引き上げ）
- **持続時間**: 60秒

### 特徴
- 初期から使える基本的な経済設備
- コストが安く、序盤から設置可能
- **60秒で消滅するため、再配置が必要**
- 元が取れるまでに50秒かかる

### 投資回収計算（v2.0）
```
初期コスト: 30
回収量: 0.6/秒 = 36/分

1回の配置（60秒）:
回収量: 0.6 × 60 = 36コスト
純利益: 36 - 30 = +6コスト

3分間の運用:
配置回数: 3回
総コスト: 30 × 3 = 90コスト
総回収: 36 × 3 = 108コスト
純利益: 108 - 90 = +18コスト

旧バージョン（永続）との比較:
旧: 25コスト → 108回収 = +83コスト
新: 90コスト → 108回収 = +18コスト
→ 経済効率が大幅にダウン（バランス調整）
```

### 運用例
- 序盤に1〜2台設置して経済開始
- 60秒後に消滅 → 再配置 or 上位設備に切り替え
- 純利益は少ないが、序盤の資金繰りには有効

## Tier2: 自動火災報知設備

### 基本情報
```javascript
{
  id: 'autoFireAlarm',
  name: '自動火災報知設備',
  type: 'yellow',
  cost: 60,
  duration: 5400, // 90秒
  icon: <BellRing size={24} />,
  desc: '【警報】コスト回復+1.2/秒＋敵感知で警告表示 [90秒]',
  effect: 'economy',
  value: 1.2,
  earlyWarning: true,
  rangeType: 'self',
}
```

### 性能詳細
- **効果1**: コスト回復量+1.2/秒（非常ベルの2倍）
- **効果2**: 敵スポーン時に画面上部に警告表示
- **コスト**: 60
- **持続時間**: 90秒

### 特徴
- 高効率な経済設備
- 早期警戒機能で戦術的な準備が可能
- **90秒と比較的長持ち**
- 中盤以降の主力経済設備

### 早期警戒システム
```
敵スポーン時の表示:
┌─────────────────────────┐
│ ⚠️ WARNING!            │
│ 新たな火災を感知！      │
│ 列 3 / タイプ: B火災   │
└─────────────────────────┘

効果:
- 敵の出現位置を事前に把握
- 防衛配置を調整できる
- 特効タワーの準備が可能
```

### 投資回収計算
```
初期コスト: 60
回収量: 1.2/秒 = 72/分

1回の配置（90秒）:
回収量: 1.2 × 90 = 108コスト
純利益: 108 - 60 = +48コスト

3分間の運用:
配置回数: 2回
総コスト: 60 × 2 = 120コスト
総回収: 108 × 2 = 216コスト
純利益: 216 - 120 = +96コスト

非常ベルとの比較:
非常ベル: +18コスト（3分間）
自動火災報知: +96コスト（3分間）
→ 自動火災報知の方が圧倒的に効率的
```

### 運用例
- 序盤の非常ベルから切り替え
- 90秒持つので2回配置で3分間カバー
- 早期警戒で戦術的優位を確保

## Tier3: 放送設備

### 基本情報
```javascript
{
  id: 'broadcastSystem',

  type: 'yellow',

  evacuationValue: 0.5,
  rangeType: 'self',
```

### 性能詳細
- **効果1**: コスト回復量+2.0/秒（自動火災報知設備の1.67倍）
- **効果2**: 避難速度+0.5人/秒（避難設備との複合効果）
- **コスト**: 100
- **持続時間**: 120秒（最長）

### 特徴
- 最高効率の経済設備
- 避難速度も上昇する複合型
- 経済と勝利条件の両立
- **120秒と最も長持ち**

### 複合効果の価値
```
経済効果:
- コスト回復 +2.0/秒
- 120秒間で +240コスト

避難効果:
- 避難速度 +0.5人/秒
- 120秒間で +60人避難

総合価値:
設置100 → 経済240 + 避難60人
= 超高効率の複合設備
```

### 投資回収計算
```
初期コスト: 100
回収量: 2.0/秒 = 120/分

1回の配置（120秒）:
回収量: 2.0 × 120 = 240コスト
純利益: 240 - 100 = +140コスト

3分間の運用:
配置回数: 2回（1回は途中で切れる可能性）
総コスト: 100 × 2 = 200コスト
総回収: 240 × 2 = 480コスト
純利益: 480 - 200 = +280コスト

他の経済設備との比較:
非常ベル: +18コスト
自動火災報知: +96コスト
放送設備: +280コスト
→ 放送設備が最も効率的！
```

### 運用例
- 中盤以降の大量コスト回復
- 避難目標達成の補助
- 経済型ビルドの最終形
- **120秒持つので実質1-2回配置で済む**

## Tier3: 消防機関へ通報する火災報知設備

### 基本情報
```javascript
{
  id: 'fireNotification',
  name: '消防機関へ通報する火災報知設備',
  type: 'yellow',
  cost: 120,
  effect: 'economyWithTransform',
  value: 1.0,
  transformDelay: 600,      // 10秒 = 600フレーム
  transformInto: 'fireEngine',
  rangeType: 'self',
  desc: '【警報】コスト回復+1.0/秒＋10秒後にポンプ車と入れ替わる',
}
```

### 性能詳細
- **効果1**: コスト回復量+1.0/秒
- **効果2**: 設置10秒後、ポンプ車に変身
- **コスト**: 120
- **特殊な持続時間**: 10秒で変身、変身後5秒で消滅

### 特徴
- 経済設備 → 攻撃設備への変身型
- 序盤は経済、後で火力に転換
- 同じマスで2つの役割を果たす
- 配置場所の戦術的重要性が極めて高い

### 変身システム
```
タイムライン:
0秒   : 消防機関通報設備を設置（コスト回復開始）
↓
0〜10秒: コスト回復+1.0/秒（累計+10コスト）
↓
10秒  : 🚒 ポンプ車に変身！
↓
10〜15秒: ポンプ車として攻撃（持続5秒）
↓
15秒  : ポンプ車消滅 → マスが空きに

合計効果:
- 経済: +10コスト獲得
- 攻撃: 5秒間の強力な縦列攻撃
- 正味コスト: 120 - 10 = 110
```

### 変身後のポンプ車
```javascript
fireEngine: {
  id: 'fireEngine',
  name: 'ポンプ車',
  type: 'purple',
  cost: 0,
  duration: 300, // 5秒
  power: 10,
  speed: 10,
  knockback: 1.5,
  damageType: 'water',
  rangeType: 'line',
  desc: '【召喚】縦1列を5秒間制圧',
}
```

### 運用例

**基本戦術:**
```
1. 敵の進行ルート上に設置
2. 10秒間のコスト回復（+10コスト）
3. 10秒後、ポンプ車に変身
4. 5秒間の強力な防衛
5. 15秒後、マスが空く
6. 空いたマスに別のタワーを配置
```

**時間差設置:**
```
0秒 : 消防機関通報①設置
5秒 : 消防機関通報②設置
10秒: 消防機関通報①がポンプ車に変身
15秒: 消防機関通報②がポンプ車に変身
      消防機関通報①のマスが空く
20秒: 消防機関通報②のマスが空く

効果:
- ポンプ車が5秒ごとに出現
- 常にどちらかが攻撃中
- マスが順番に空く
```

## 経済型ビルドの基本理論（v2.0）

### 投資タイミング

```
序盤（0〜60秒）:
優先度: 永続設備 > 経済設備
理由: 永続設備は再配置コストがかからない

推奨:
誘導灯（60コスト / 永続）→ HP回復+避難
連結送水管（60コスト / 永続）→ 周囲バフ
非常ベル × 1台（30コスト / 60秒）→ 経済開始

中盤（60〜120秒）:
優先度: 中コスト警報 + 防衛強化
理由: 敵が強化される前に経済力確保

推奨:
自動火災報知設備（60コスト / 90秒）
非常ベルの再配置 or 切り替え

終盤（120〜180秒）:
優先度: 高コスト警報 + 最終防衛
理由: 残り時間が短いが高効率設備なら回収可能

推奨:
放送設備（100コスト / 120秒）→ 経済+避難
消防機関通報（120コスト）→ ポンプ車で緊急防衛
```

### 投資回収の損益分岐点

```
設備名          | コスト | 回収/秒 | 持続 | 1回の純利益 | 3分純利益
----------------|--------|---------|------|-------------|----------
非常ベル        | 30     | 0.6     | 60秒 | +6          | +18
自動火災報知    | 60     | 1.2     | 90秒 | +48         | +96
放送設備        | 100    | 2.0     | 120秒| +140        | +280

結論:
- 上位設備ほど純利益が高い
- 放送設備が最も効率的
- ただし初期コストが高いので、序盤は非常ベルから
```

## まとめ

### 警報設備の特徴（v2.0）
✅ **経済力の源** - コスト回復でゲームを有利に
✅ **有限の持続時間** - 60〜120秒で消滅
✅ **上位ほど効率的** - 放送設備が最高効率
✅ **複合効果** - 放送設備は避難も加速

### 持続時間まとめ
| 設備 | コスト | 持続時間 | 3分純利益 |
|------|--------|----------|-----------|
| 非常ベル | 30 | 60秒 | +18 |
| 自動火災報知 | 60 | 90秒 | +96 |
| 放送設備 | 100 | 120秒 | +280 |
| 消防機関通報 | 120 | 特殊 | 変身型 |

### v2.0での変更点
- **非常ベル**: コスト25→30、持続時間60秒（経済効率を調整）
- **全設備**: 持続時間を設定（消滅→再配置のサイクル）
- **バランス**: 上位設備ほど効率が良い設計に

### 推奨ビルド
- **初心者**: 自動火災報知設備で安定経済
- **中級者**: 放送設備で経済+避難の両立
- **上級者**: 消防機関通報で経済+ポンプ車の複合運用

### 注意点
⚠️ 警報設備も消滅する（再配置コストを計算）
⚠️ 序盤の投資ほど総回収量が多い（早期投資推奨）
⚠️ 放送設備は120秒持つので、タイミングを考えて配置
